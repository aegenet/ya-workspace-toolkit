#  Yawt

Yet Another Workspace Toolkit - The purpose of this package is to provide a set of tools to help you manage your workspace.

## 💾 Installation

```shell
npm i @aegenet/yawt@~1 -D
# or
yarn add @aegenet/yawt@~1 -D
```
